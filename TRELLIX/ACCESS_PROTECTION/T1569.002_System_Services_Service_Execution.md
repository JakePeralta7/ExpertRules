# T1569.002: System Services: Service Execution

## Author
Trellix

## Description
This Expert rule detects when user is trying to create services.

## Rule Class 
Process

## Rule TCL
```tcl
Rule {
    Target {
        Match PROCESS {
            Include OBJECT_NAME { -v "sc.exe" }
            Include PROCESS_CMD_LINE { -v "**create* binPath=** }
            Include -access "CREATE"
        }
    }
}
```
